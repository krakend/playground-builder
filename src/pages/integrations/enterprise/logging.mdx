## About this demo
Kibana stores KrakenD logs and allows analysis.

Access the demo: [http://localhost:5601/](http://localhost:5601/)

**NOTE**: To import a Kibana dashboard with some valuable metrics, run in the console `make elastic` first.

Then you can click on the burger menu and go to Analytics > Dashboards
