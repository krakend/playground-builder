## About this demo
Postman collections can be generated with `krakend generate openapi.`

## Command
While the docker compose system is running, you can call the `krakend generate command` from the krakenD container or a new process. For instance:
```bash
docker-compose exec krakend_ee krakend generate postman -c /etc/krakend/krakend.json -o /etc/krakend/demo/postman.json
```
