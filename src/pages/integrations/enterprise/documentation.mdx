## About this demo
From an existing KrakenD API Gateway specification you can create automatically its OpenAPI documentation with `krakend openapi export.`

But if you have an existing OpenAPI specification you can create a KrakenD Configuration from it.

![OpenAPI generation screenshot](https://www.krakend.io/images/documentation/openapi-generator.png)

## Export to OpenAPI
```bash
docker-compose exec krakend_ee krakend openapi export -c /etc/krakend/krakend.json -o /etc/krakend/demo/swagger.json
```

## Import to OpenAPI
If you have an `openapi.yaml` file you can do:
```bash
docker-compose exec krakend_ee krakend openapi import -s openapi.yaml -o newkrakend.json
```
