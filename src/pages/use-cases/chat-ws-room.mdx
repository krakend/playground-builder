## Chat room using WebSockets
The endpoint `/chat/ws/{room}` creates a WebSockets connection to the selected `{room}`. KrakenD upgrades the connection and proxies it to the backend broker; messages flow bidirectionally through the gateway.

Choose a room name and connect the two clients below to create two independent WS connections. The same dual-client UX is available in the SPA's WebSocket Chat tab.
