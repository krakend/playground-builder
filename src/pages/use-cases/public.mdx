## A public endpoint

This endpoint has **no authentication** declared and is open to any caller. KrakenD forwards the request straight to the backend without any JWT validation, API-key check or policy enforcement.

Public endpoints are the right choice for content that should be reachable without credentials — health checks, landing pages, login flows or any anonymous browsing path. Pair them in the same gateway with the [`/private/*` endpoints](/demo/use-cases/private-moderate) to compare what KrakenD enforces when authentication IS declared.

}
